stupid currency converter
A simple, no-frills, mobile-first currency converter built for speed. Designed to be added to your phone's home screen as a web-app for quick access.
!
This is just a single HTML file, so you can host it for free on GitHub Pages, Netlify, or any static site host.
Features
 * Live Rates: Pulls live exchange rates from the free frankfurter.app API.
 * All Currencies: Search and select from all available global currencies.
 * Spin to Win (or Lose): An interactive "knob" to increase or decrease the amount by dragging up and down.
 * Quick Controls: Easily swap 'From' and 'To' currencies or reset the amount to 1.
 * Mobile First: Clean, responsive design that works perfectly on a phone.
Tech Stack
 * HTML
 * Tailwind CSS (loaded via CDN)
 * Vanilla JavaScript (ES6+)
No build step, no frameworks, just one file.
How to Use as a Phone Shortcut
 * Host It: Host the index.html file (e.g., using GitHub Pages).
 * Open in Browser: Open the live URL on your phone's browser (Safari on iOS, Chrome on Android).
 * Add to Home Screen:
   * iOS (Safari): Tap the 'Share' icon and select 'Add to Home Screen'.
   * Android (Chrome): Tap the three-dot menu and select 'Add to Home screen' or 'Install app'.
This will add an icon to your home screen that launches the converter like a native app.
